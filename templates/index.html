Vulnerable Endpoints

Classic SQL Injection — Authentication Bypass via login form string concatenation.

Critical Data Extraction

Error-Based SQL Injection — Trigger SQL errors to reveal database structure.

Boolean-Based Blind SQLi — Infer data by observing True/False page differences.

High Blind
