🔬 Why Stacked Queries Are Dangerous
  • Most SQL injection only reads data — stacked queries allow data modification
  • Attackers can create persistent backdoor accounts even after the original vulnerability is patched
  • Can be used for privilege escalation, data destruction, or planting evidence
  • Not all database drivers support stacked queries (SQLite executescript() does)
  • In MySQL with mysqli_multi_query() and MSSQL with ; separators
